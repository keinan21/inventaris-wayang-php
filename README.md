# 🗃️ Project Management Box Wayang

Aplikasi manajemen proyek sederhana untuk mencatat pembuatan dan perawatan Box Wayang. Dibangun menggunakan **PHP Native** dengan pendekatan **OOP (Object-Oriented Programming)** untuk belajar dasar-dasar manipulasi database.

## 🚀 Fitur Utama
- **Create**: Menambah data proyek wayang baru melalui form.
- **Read**: Menampilkan daftar proyek dalam tabel yang responsif.
- **Update**: Mengubah data proyek (Nama & Tahun) menggunakan modal `<dialog>` dan JavaScript.
- **Delete**: Menghapus data proyek dengan konfirmasi keamanan.

## 🛠️ Teknologi yang Digunakan
- **Bahasa**: PHP 8.x (OOP Style)
- **Database**: MariaDB (MySQL) via XAMPP
- **Styling**: [Water.css](https://watercss.kualo.net/) (Lightweight CSS Framework)
- **Icons**: Bootstrap Icons
